| BUG-001 | numpy模块缺失导致头像生成任务失败 | 头像生成功能完全不可用 | 🔴 高 |

**问题详情**:
- **位置**: `server/app/routers/avatar.py:51`
- **错误信息**: `No module named 'numpy'`

**修复建议**:
```bash

source venv/bin/activate
pip install numpy pillow
```
