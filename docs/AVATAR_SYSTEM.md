## 系统概述

像素头像动画系统为用户提供个性化像素风格头像，支持三种情绪动画（开心/难过/激动），并根据用户心情值自动切换显示情绪。

### 核心特性

- **AI 生成**: 使用火山引擎 Doubao SeeDream 模型生成像素风格动画
- **多情绪支持**: 一次生成 3 种情绪（happy/sad/excited）的 GIF 和 PNG
- **心情驱动**: 根据用户 mood 值自动推荐和切换情绪
- **异步处理**: 后台任务处理，支持进度查询

                                        ┌────────┴────────┐

                                │ mood <= 70  │   │ mood > 70   │
                                │→ new="happy"│   │→ new="excited"│
                                └─────────────┘   └─────────────┘
                                                 │

                                        ┌─────────────────┐

## 数据模型

### User 表字段（头像相关）

```python
class User(Base):
    # 基础信息
    user_id: str          # 用户ID
    mood: int             # 心情值 (0-100)
    
    # 头像路径
    avatar_happy_gif: str     # 开心动画
    avatar_happy_png: str     # 开心静态
    avatar_sad_gif: str       # 难过动画
    avatar_sad_png: str       # 难过静态
    avatar_excited_gif: str   # 激动动画
    avatar_excited_png: str   # 激动静态
    
    # 当前状态
    current_emotion: str      # 当前情绪: "happy" | "sad" | "excited"
```

server/
