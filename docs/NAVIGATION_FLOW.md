```
│                                                                                 │

```

## 组件层级结构

```
App
└── QueryClientProvider
    └── RouterProvider
        └── HomePage "/"
            ├── 头像区域
            │   └── 情绪头像 (mood 驱动)
            ├── 属性条
            │   ├── Power 条
            │   └── Mood 条
            ├── 标签区
            │   └── UserTags
            ├── FIGHT 按钮
            │   └── 面试选择面板
            │       ├── 文字面试 → /interview
            │       ├── 语音面试 → /voice-interview
            │       └── 面试历史 → /interviews/history
            ├── 任务面板
            │   └── DailyTaskList
            ├── 底部导航
            │   ├── 朋友 → /friends
            │   ├── 打卡 → /checkin
            │   ├── 树洞 → /tree-hole
            │   ├── 简历 → /resume
            │   ├── 成就 → /achievements
            │   └── 历史 → /interviews/history
            └── 引导弹窗
                └── 新用户提示 → /survey
```
